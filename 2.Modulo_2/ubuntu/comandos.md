## Comandos básicos en Ubuntu

### Cambiar dir
```
cd carpeta
```
### Carpeta superior
```
cd ..
```
### Ver dir actual
```
pwd
```
### Ver archivos en ubicación actual
```
ls
```
[Más comandos](https://www.guia-ubuntu.com/index.php/Comandos)

## Ejercicio
Realizar estos comandos y razonar lo que vemos:
```
cd /home/usuario 
```
o si no estamos logeados en un usuario:
```
cd /root
````
